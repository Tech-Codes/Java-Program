# Snake Game

## About

This is a simple Snake Game made using Java Swing.

The snake moves using arrow keys and eats apples to grow longer.

---

# Features

* Snake movement
* Apple generation
* Collision detection
* Game Over screen
* Keyboard controls

---

# Requirements

Install Java JDK 8 or higher.

Check Java version:

```bash
java -version
```

---

# Project Structure

SnakeGame/
│
├── Board.java
└── README.md

---

# Compile the Program

Open terminal in project folder and run:

```bash
javac Board.java


---

# Run the Program

```bash
java Board
```

---

# Controls

| Key | Action     |
| --- | ---------- |
| ↑   | Move Up    |
| ↓   | Move Down  |
| ←   | Move Left  |
| →   | Move Right |

---

# Game Rules

* Eat red apples to increase score.
* Do not touch walls.
* Do not collide with your own body.

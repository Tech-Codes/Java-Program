# Java Number Guessing Game

A simple console-based Number Guessing Game built using Java.

## About the Project

This game generates a random number between 1 and 100.  
The player must guess the correct number with hints provided after each attempt.

- "Too High!" → Guess is greater than the number
- "Too Low!" → Guess is smaller than the number
- "Correct!" → Player wins the game

## Features

- Random number generation
- User input handling
- Attempt counter
- Interactive console gameplay
- Beginner-friendly Java project

## Technologies Used

- Java
- Scanner Class
- Random Class
